# ChatGPT-DateConversion
This repository accompanies the article “ChatGPT und die Datierungskonventionen deutscher Regestenkunst”. Using 
[prompt1](./prompt1.txt) and the modified version 
[prompt2](./prompt2.txt), ChatGPT-4o and ChatGPT-4o mini were each asked five respective ten times to convert verbal dates into a numeric format. 
The results ([gold_vs_results](./gold_vs_results.tsv)) can be visualized with 
[visualize_prompt_results](./visualize_prompt_results.ipynb). For the article, the figures 
[comparison_early](./comparison_early.png) and 
[comparison_late](./comparison_late.png) 
were subsequently enhanced with structuring lines.  
Finally, [convertRegestaDatetoISO](./convertRegestaDatetoISO.ipynb) is a program that was likewise created in dialogue with ChatGPT and enables the conversion of dates using regular expressions (RegEx).
